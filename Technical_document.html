Introduction to Python

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

It is used for:

  • web development (server-side),
  • software development,
  • mathematics,
  • system scripting.
Python Basics

The simplest functions within python include creating variables, and printing statements. Let's begin with a simple example for practice. We will start by creating a variable and printing it to see the outcome.

greeting = "Hello World"

print(greeting)

# Output: Hello World

As you can see, the variable greeting was created and assigned the value "Hello World". The print statement was used to display the value of the variable. We will go further in-depth into variables in the next section.

Python Variables

Variables are some of the most important functions you will learn to implement, since nearly everything you create using python will include them. You can assign strings and numbers to variables, then these variables can be used to complete various functions, without making your code look ugly. Variables can be assigned simply be providing a word or letter (no numbers), followed by an "=", then the string or number (or any other type) you would like to assign to that variable. Here are a few examples:

x = 5

y = "John"

print(x)

print(y)

# Output: 5

# Output: John

As you can see, the variable x was assigned the value 5, and the variable y was assigned the value "John". The print statement was used to display the values of the variables. You can also change the value of variables without consequence (unless the variable has been used in a function beforehand), since python executes cells from top to bottom in order. For example:

x = 5

print(x)

# Output: 5

x = "woah"

print(x)

# Output: woah
Python Strings

Strings in python are surrounded by either "" or '', and a string literal can be displayed using the print function.

print("Hello")

# Output: Hello

Strings can be assigned to variables, and you can concatenate strings by using the "+" operator. For example:

a = "Hello"

b = "World"

c = a + b

print(c)

# Output: HelloWorld
Python Numbers

There are three number types in python:

  • int, or integer, is a whole number, without decimals.
  • float, or 'floating point number' is a number containing one or more decimals.
  • complex, written with a 'j' as the imaginary part.

Here are some examples:

x = 1 # int

y = 2.8 # float

z = 1j # complex
